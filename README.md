# SC9 Photo Book 2025

This repository currently contains a minimal landing page skeleton for the SC9 Photo Book 2025 project. Use it as the fresh starting point for rebuilding the experience.

## Structure
- `index.html` – root landing markup.
- `style.css` – shared styles for the landing card and layout.
- `script.js` – lightweight fade-in behavior.

Feel free to layer new sections, pages, or build tooling on top of this clean baseline.
